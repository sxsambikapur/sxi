## üìñ About The Project
This is a modern, responsive website designed for the **St. Xavier's Group of Institutions** in Ambikapur, Chhattisgarh. The website showcases the various educational units under the group, including the School, College, Nursing College, and Hostel.

## ‚ú® Key Features
* **Modern UI/UX:** Features a "Glassmorphism" navigation bar and card layouts.
* **Animations:** Smooth scroll reveals (`IntersectionObserver`), parallax hero effects, and hover transitions.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens.
* **Performance:** Includes a custom Javascript preloader for smooth asset loading.
* **Jesuit Identity:** Dedicated sections highlighting Core Jesuit Values (Magis, Cura Personalis).
